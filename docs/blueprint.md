# **App Name**: TechXera Campus

## Core Features:

- Home Page UI: Displays the hero section with an animated tech background, tagline 'Empowering Students Through Technology', and includes navigation for Login and Check Results. Incorporates smooth scroll animations.
- Student Login & Dashboard: Provides a secure login form for students using ID and password. Authenticated students access a personalized dashboard displaying marks, grades, and a performance chart with animated progress bars and cards.
- Study Materials & Resources: Presents previous questions, notes, and various study resources. Users can filter by subject or semester and download materials with engaging hover animations on buttons.
- Important Notices Board: Features a dynamic notice board with notices displayed in a card layout. Urgent notices are visually highlighted in red, with fields for date, title, and description.
- Admin Content Management: A UI for administrators to upload, edit, and delete student results, notices, and study questions. Presented via table views with basic CRUD (create, read, update, delete) operations.

## Style Guidelines:

- Primary interactive color: A vivid medium violet-blue (#7D6BDB) to convey technology and empowerment. It will be used for buttons, interactive elements, and accents.
- Background color: A very light purplish-gray (#E0E0EB), providing a clean, bright base that allows interactive elements to stand out, contributing to a modern, light-and-dark mixed theme.
- Accent color: A vibrant sky blue (#4FB2EA) to provide visual highlights, draw attention, and complement the primary color while fitting the requested 'blue, purple, white accents'.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy aesthetic. Body font: 'Inter' (sans-serif) for clean readability and a contemporary, objective feel.
- Utilize a minimalist, consistent line-art style icon set that aligns with a clean UI and modern aesthetic.
- Employ a responsive grid-based layout for 'student-portal style' sections, favoring card-based structures for notices and study materials. Integrate glassmorphism and subtle 3D effects for depth.
- Incorporate smooth transitions, micro-animations on interactive elements (buttons, cards, menus), loading animations, hover effects, smooth scroll animations, and an animated tech background on the home page.
